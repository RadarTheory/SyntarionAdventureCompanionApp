## Built With

Syntarion stands on excellent open-source software and services:

- **React** — UI library (MIT License)
- **Vite** — build tooling (MIT License)
- **Supabase** — database, authentication, and storage
- **Vercel** — hosting and deployment
- **Google Gemini** — powers the Scribe and session synopsis features
- **Groq (Whisper)** — powers voice transcription

Each open-source dependency is governed by its own license; see the repository for the full dependency list.
