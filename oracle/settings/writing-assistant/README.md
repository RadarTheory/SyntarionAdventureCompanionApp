# Oracle Writing Assistant Settings

This folder contains the configuration Oracle needs to behave like a writing assistant for Soteria and the Soterian Continuum.

The structure mirrors the current Google Drive organization:

- `1 Soteria`
- `1 Lore`
- `2 Maps`
- `3 Player Characters`
- `4 Official Symbols`
- `5 Playable Classes`
- `6 Playable Races`
- `SOTERIAN PLAYER HANDBOOK | 2026-2027`
- `1 The Soterian Continuum | The Sovereign Kingdom`
- `7 The Soterian Continuum | Short Stories`

Oracle should use these settings to decide what to retrieve, how to cite it, and what kind of output is allowed.

The core rule remains: no AI-generated writing, fact, relationship, location note, or lore update becomes canon without explicit approval.
