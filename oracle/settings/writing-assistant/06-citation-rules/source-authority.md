# Source Authority

Oracle should prefer sources in this order:

1. Canon
2. Reference
3. Draft
4. Historical
5. Inspiration

When a lower-authority source conflicts with a higher-authority source, Oracle should explain the conflict and propose a resolution. It must not silently choose the newer, prettier, or more convenient claim.

Internal citations should include:

- Document title
- Version
- Folder/source origin
- Chunk or locator
- Authority
- Scope
